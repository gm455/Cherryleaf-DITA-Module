Thinking Mobile Creator Overview

The ThinkingMobile Creator enables you to create ThinkingMobile files for your WAP site.

ThinkingMobile files can react to criteria such as:

  • Date
  • Time
  • Age
  • Mobile device

This means that you can configure and optimise images to be displayed under specific criteria. You can show specific images on special occasions and make sure that the image is viewable on a specific mobile device.You can also pass your own criteria into ThinkingMobile files

About dynamic content

ThinkingMobile Creator allows you to create dynamic content for web sites that are viewed on mobile devices.

This means that you can display content which varies according to criteria that you choose. For example, you can:

  • Optimise text and images to be viewed on different types of device
  • Tailor text and images to age groups
  • Show different content on specific days or dates
  • Personalise content

Creating a ThinkingMobile file from scratch

You can create a ThinkingMobile file from scratch or by using the Wizard. ThinkingMobile files enable you to set up images and text to be displayed on different devices and under different circumstances.

For example, you can choose an image that will only be displayed on a particular date.

To create a ThinkingMobile file from scratch

  1. From the File menu, choose New.
    The New ThinkingMobile file dialog box is displayed.
  2. Choose the Start from scratch with a blank file radio button and click OK.
    You are returned to the main screen.
You must now set default images for different devices

Opening a ThinkngMobile file

ThinkingMobile files enable you to set up images and text to be displayed on different devices and under different circumstances.

To open a ThinkingMobile file

  1. From the File menu, choose Open.
    The Open file... dialog box is displayed.
  2. Select the file that you want to open and click Open.
    You are returned to the main screen and your file is displayed.
    Note: You can only open ThinkingMobile files. These files have a .tmf extension

Setting an image for a device

How to set an image for a device.

Before you can do this you must create or open a ThinkingMobile file.
You can specify images to be displayed on the following devices:
  • phone
  • portrait
  • landscape

You can also specify images to be displayed on each of the above devices under different circumstances.

  1. In the left-hand pane of the main screen select the device.
    The device is displayed in the right-hand pane.
  2. Click
    The Image Properties dialog box is displayed.
  3. Click Open file....
    The Open file... dialog box is displayed.
  4. Browse for the image and click Open.
    Note: You can only choose an image format that is supported by ThinkingMobile.
    The image is displayed in the device in the right-hand pane of the main screen.
